
# 🦠 COVID-19 India Dashboard

## 📌 Overview

The **COVID-19 India Dashboard** is an interactive Power BI project that visualizes COVID-19 statistics across Indian states and union territories. It provides insights into confirmed cases, active cases, recoveries, deaths, and population through interactive charts and filters.

---

## 📊 Dashboard Features

- Total Cases
- Total Active Cases
- Total Recovered
- Total Deaths
- Total Population
- State

---

## 📈 Visualizations

- Total Cases by State
- Active Cases by State
- Total Deaths by State
- Population by State/UT
- Active Rate Gauge
- Death Rate Gauge
- Case Distribution (Recovered, Active & Deaths)

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

---

## 📂 Dataset

The dataset includes COVID-19 information such as:

- State/UT
- Confirmed Cases
- Active Cases
- Recovered Cases
- Deaths
- Population

---

## 🎯 Key Insights

- Compare COVID-19 cases across Indian states.
- Monitor active, recovered, and death cases.
- Analyze state-wise population and case distribution.
- Track COVID-19 trends using interactive visuals.
